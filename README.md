# MED-VIP (Vertically Intergrated Project)

Medical Device VIP is a Virginia Commonwealth University project group that specializes in Medical Devices. There are multiple division of this project group.

This Repository focuses on the division of hardware infiltration. The goal for this repo is to use this as a source of information for embeded systems knowledge. Users of this repo will learn the basics from how an operating system works, to intermediate concepts like bash scripting, to high level firmware extraction methods for certain targeted devices.
